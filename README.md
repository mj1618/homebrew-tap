# Homebrew Tap for RunParticle

Install the `particle` CLI via Homebrew.

## Install

```bash
brew install mj1618/tap/particle
```

Or add the tap first:

```bash
brew tap mj1618/tap
brew install particle
```

## Upgrade

```bash
brew upgrade particle
```

## About

[RunParticle](https://runparticle.com) — instant cloud VMs from your terminal.
